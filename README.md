# Simulador de Potencial de Concessões Florestais

Este projeto tem como objetivo calcular os impactos socioambientais e os investimentos necessários com base na expansão da área concedida (em hectares) para concessões florestais.

## Funcionalidades
- Cálculo do potencial de carbono retido/evitado por hectare ao ano
- Estimativa de empregos diretos e indiretos gerados
- Projeção de retorno financeiro anual
- Cálculo do investimento necessário para gestão do Serviço Florestal

## Como usar
1. Execute o script Python `impacto_concessoes.py`.
2. Informe a meta de área em hectares quando solicitado.
3. O programa exibirá os resultados dos cálculos diretamente no console.

## Estrutura do projeto
- `main.py`: Exemplo de simulação básica
- `impacto_concessoes.py`: Função principal para cálculo dos impactos
- `.gitignore`: Arquivo para ignorar arquivos e pastas desnecessários no versionamento

## Requisitos
- Python 3.7 ou superior
